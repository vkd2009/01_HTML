style 태그 내용 적용 확인

block 형식

        - 화면(공간)을 수직 분할
         (세로 화면을 가로로 잘라서 층(행)을 나눔)

        - 한 층(한 행)을 모두 차지함
        
        - width, height 와 같은 크기 지정 속성 사용 가능


        - block 형식의 태그
         h1~h6(제목) , p/pre(문단) , hr(수평선), 
         div(영역 분할), 시맨틱 태그(div 태그에 이름 넣은 태그)

h3는 block 형식 태그 입니다

        block 형식은 작성된 내용이 짧더라도 
        한 줄을 모두 차지하려는 성질 때문에 
        다음 태그 내용은 다음 줄에 작성됩니다
첫 번째 div
두 번째 div
세 번째 div

inline 형식

        - 화면을 수평 분할
          (가로 화면을 세로로 잘라서 호(열)을 나눔)

        - 한 층(행)을 모두 차지하지 않음
          -> 내부에 작성된 내용 크기 만큼만 차지

        - width, height 와 같은 크기 지정 속성 사용 불가!!! 
        
        -inline 형식의 태그
        b(굵게), i(기울이기), mark(글배경 칠하기),
        span(영역 분할) 등

inline 형식은 형을 분할하는 성직을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다
첫 번째 두 번째 세 번째

영역 지정 방식의 차이점

  • block 형식 : 사각형 박스 형태로 영역을 지정
  • inline 형식 : 내부에 작성된 내용(content) 단위로 영역을 지정
