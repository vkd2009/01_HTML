a(anchor: 연결, 고정) 태그

      - 현재 HTML 문서와 다른 HTML 문서를 연결 시켜
        페이지를 이동할 수 있게 만드는 태그에
      - a 태그로 감싸진 글/내용 == 하이퍼 텍스트(하이퍼 링크)

      - 하이퍼 텍스트는 다른 문서와 구분되는 HTML만의 핵심 기능

    지정된 id 속성을 가진 요소(태그)로 이동

    하나의 HTML 문서 내에서 
    각각의 HTML 요소(태그)를 구분하는 식별 값
    -> 중복되는 ID가 존재하면 안된다!
