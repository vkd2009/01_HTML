글자에 적용되는 기타 태그들

b 태그 : 단순히 글자를 굵게 표시하는 태그
i 태그 : 단순히 글자를 기울여 표시하는 태그

strong 태그 : 글자(단어)를 강조하는 용도의 태그
em 태그 : 글자(문장)를 강조하는 용도의 태그 

            strong, em 태그는 웹 접근성을 위해 사용되는 태그이다.

            웹 접근성이란?
            일반인 뿐만 아니라 장애인, 고령자들을 위해
            웹 사이트에서 제공하는 서비스를 모두 이용할수 있도록 보장한 것
            (의무사항)

            ex) 스크린 리더
        
            mark 태그 : 형광펜 효과 를 나타내는 태그

            u 태그 : 밑줄을 긋는 태그

            s 태그 : 글자 가운데 선을 긋는 태그(취소선, 잘못 작성된 것을 표시)

            del 태그 : 글자 가운데 선을 긋는 태그(삭제선, 이제 없는 것을 표시)

            

            small 태그 : 글자를 작게 표시하는 태그

            sup 태그(윗첨자) : x3 

            sub 태그(아랫첨자) : log2

            abbr(abbreviation: 약어) 태그 : 마우스 오버 시 툴팁 표시
